# Security Policy

## Purpose
This repository contains **defensive cyber threat intelligence (CTI) research** based exclusively on **open-source intelligence (OSINT)**.

The purpose of this project is to demonstrate **ethical threat analysis, reconnaissance awareness, and defensive security practices** aligned with real-world SOC and Blue Team operations.

---

## Scope of Work
The following activities were intentionally performed:

- Passive OSINT collection only
- Analysis of **publicly accessible information**
- Threat modeling and MITRE ATT&CK mapping
- Defensive recommendations and risk assessment

The following activities were **explicitly excluded**:

- No vulnerability scanning
- No exploitation attempts
- No credential testing
- No social engineering
- No interaction with live systems
- No bypassing of access controls

---

## Responsible Use
This repository **must not be used** for:
- Unauthorized access
- Exploitation of systems or users
- Harassment, impersonation, or fraud
- Any illegal or unethical activity

All findings are presented in a **defensive and educational context only**.

---

## Reporting Concerns
If you believe any content in this repository:
- Exposes sensitive information unintentionally, or
- Could be misinterpreted or misused,

please contact the repository owner privately before public disclosure.
